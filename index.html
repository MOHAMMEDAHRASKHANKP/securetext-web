SecureText — Client-side Web

AES-GCM (PBKDF2) & RSA-OAEP in the browser — keys never leave your device
Format: {"salt":"hex","iv":"base64","cipher":"base64"}
RSA-OAEP SHA-256, 2048-bit
Warning: Keep private PEM safe. If you export it unencrypted, anyone with the file can decrypt messages.
Encrypted file will be downloaded as .securetext (JSON wrapper with base64 data)
Decrypted file will be downloaded with original filename.
Notes: This app uses PBKDF2 in-browser. High iteration counts increase security but also CPU usage. RSA key ops are slow on low-end devices. Keys and data never leave your browser unless you choose to download or paste them somewhere.
